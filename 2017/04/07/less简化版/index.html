    less是什么

    Less 是一门 CSS 预处理语言，它扩展了 CSS 语言，增加了变量、Mixin、函数等特性

    less有什么优势

    使 CSS 更易维护和扩展。

    Less 可以运行在 Node 或浏览器端。

    less怎么使用

    关于less使用，在前期一定要规划好，哪里用什么颜色，也就是相当于一套规范，比如导航用是什么字体，什么颜色，标题用什么颜色，内容用什么颜色，这都要很好空子
